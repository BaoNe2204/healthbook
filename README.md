# HealthBook Project

This is a Monorepo containing both the Frontend (Mobile App) and the Backend API for the HealthBook project.

## Directory Structure

- `mobile-app/` - The Android mobile application (Frontend). Built with Kotlin and Jetpack Compose.
- `backend-api/` - The Backend API server (e.g. Node.js, Spring Boot, .NET).
- `database/` - SQL scripts and database migrations.
- `docs/` - Project documentation, API specs, and system designs.
- `postman/` - Postman collections for testing the API.
